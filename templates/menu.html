Info

Esta página proporciona una interfaz de usuario para administrar tareas dentro de un proyecto, organizarlas en etapas y ver estadísticas del proyecto.

El administrador también pueden crear nuevas tareas y administrar miembros del proyecto. Las tareas en el tablero kanban se pueden arrastrar, lo que permite a los usuarios cambiar sus etapas.

La parte de tareas sirve como interfaz de usuario para administrar tareas personales, incluida la creación de nuevas tareas, marcar tareas como completadas y eliminar tareas.
